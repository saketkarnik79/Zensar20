Introduction to NodeJs
	What is NodeJS?
		It is an open-source, cross-platform JavaScript runtime environment that allows us to run JavaScript outside the browser, primarily on the server side.
	
		Notes:
			NodeJS is not a framework.
			NodeJS is not a programming language either.
			NodeJS is just a runtime built on Google's V8 JavaScript engine.
		Why NodeJS was primarily created?
			It was built in 2009 to solve performance issues in existing traditional web servers.
			Traditional server models
				Each request creates/allocates a new thread.
				Threads consume memory.
				Poor scalability under heavy load.
			How NodeJS solves this problem?
				Uses event-driven, non-blocking I/O.
				Runs single-threaded but still handles thousands of clients efficiently.
				Event loop takes responsibility for it.
				Ideal for real-time & scalable application.
			Key goal of NodeJS:
				Handle many concurrent connections with minimal resources.
			Key features of NodeJS
				Single Threaded & Event-Driven Architecture
				Non-Blocking I/O
				Fast Execution
				Highly Scalable
				Cross-Platform
			Core components of NodeJS:
				V8 engine
				Event Loop

			Advantages of NodeJS
				One language for frontend & backend
				Huge ecosystem (NPM)
				High Performance
				Scalable
				Active community
			Limitations of NodeJS
				Not good for CPU-intensive tasks
				Single-threaded (needs clustering for very heavy workloads)
				Call hell(mostly solved using async/await)
			When should we use NodeJS?
				If:
					we need high concurrency
					our app is I/O heavy
					we want real-time features
					we prefer JavaScript everywhere
			When to avoid NodeJS?
				If:
					our app is CPU-intensive.
					we need heavy computation.

		function readFileAsync(path) {
		  return new Promise((resolve, reject) => {
			require("fs").readFile(path, "utf8", (err, data) => {
			  if (err) reject(err);
			  else resolve(data);
			});
		  });
		}
		
		
		console.log("Start");

		readFileAsync("data.txt")
		  .then((data) => {
			console.log("File:", data);
		  })
		  .catch((err) => {
			console.error("Error:", err);
		  });

		console.log("End");
		
		
		getUserById(1)
		  .then(user => getOrders(user.id))
		  .then(orders => getPayments(orders))
		  .then(payments => console.log(payments))
		  .catch(err => console.error(err));
		  
		  
		  
		async function processUser(id) {
		  try {
			const user = await getUserById(id);
			const orders = await getOrders(user.id);
			const payments = await getPayments(orders);
			console.log(payments);
		  } catch (err) {
			console.error("Error:", err);
		  }
		}

		processUser(1);
